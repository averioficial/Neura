Esta Política de Privacidade explica como o bot Neura coleta, armazena e utiliza informações dentro do Discord.

1. Informações Coletadas

O Neura pode coletar e armazenar os seguintes dados:

IDs de usuários

IDs de servidores

IDs de canais

Configurações de economia e perfil dentro do bot

Logs de comandos para prevenção de abusos

Nenhum dado pessoal sensível é coletado (como nome real, e-mail, localização, etc.).

2. Como os Dados São Usados

Os dados coletados são usados apenas para:

funcionamento correto dos sistemas de moderação

economia e progressão de usuários

comandos úteis e funções de diversão

melhoria contínua do bot

proteção contra spam e abusos

3. Compartilhamento de Dados

O Neura não compartilha dados com terceiros.
Nenhum dado é vendido, alugado ou repassado a outras plataformas.

4. Armazenamento e Segurança

Os dados são armazenados de forma segura.

São utilizados apenas pelo bot durante o funcionamento.

Você pode solicitar a remoção dos seus dados entrando em contato com o desenvolvedor.

5. Exclusão de Dados

Se você quiser excluir seus dados do banco do Neura, basta entrar em contato com o desenvolvedor e fornecer seu ID do Discord.

6. Alterações na Política

A política pode ser modificada a qualquer momento.
Continuações no uso do bot significam que você aceita as mudanças.
